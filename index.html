    How to Use

    1. Type a keyword (e.g., "gfg" for GeeksforGeeks, "docs" for Google Docs).

    2. The search bar will instantly show suggestions as you type.

    3. Press Enter to navigate to the first suggestion, or click on any suggestion in the list.

    4. Use the category buttons to filter links.

    Shortcut Keys

    • Cmd/Ctrl + K: Focus the search bar
    • Tab: Cycle through category buttons
    • Shift + Tab: Cycle backwards through category buttons
    • Enter: Activate selected category button or open selected link
    • ⬆ / ⬇: Navigate through suggestions
    • ⬅ / ➡: Navigate through search engine buttons (when no results)
    • Enter: Open selected link/search in selected engine
    • Esc: Clear search and suggestions
    • Cmd/Ctrl + /: Toggle this help panel
    • Cmd/Ctrl + F: Focus feedback form
    • Cmd/Ctrl + S: Focus suggestion form
